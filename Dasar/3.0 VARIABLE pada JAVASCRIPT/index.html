Variable consists of

  • Declaration is registering a variable in the appropriate scope
  • Initialization is to provide memory for variables
  • Assignment is assigning a specific value to a variable

Variable Writing Rules

  1. Variable cannot use spaces
  2. Variable cannot start with a number
  3. variable cannot use keyword & reserverd word(break, case, catch, class, const, continue, debugger, default, delete, do, else, enum, export, extends, false, finally, for, function, f, implements, import, in, instance, of, interface, let, new, null, package, private, protected, public, return, static, super, switch, this, throw, true, try, typeof, var, void, while, with, yield)

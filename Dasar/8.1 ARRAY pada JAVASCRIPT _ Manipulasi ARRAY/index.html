Array Manipulation

  • Add element to an array
  • Delete element in array
  • Display the entire array

Method in Array

  • length : to find out the number of elements in the array
  • join : a method to concatenate the entire array and convert it into a string
  • push(mendorong masuk) : add an array element to the end of the array
  • pop(memecahkan atau menghilangkan) : remove the last element from an array
  • unshift(menggeser kedalam) : exactly same as push, but it works for the first element in the array
  • shift(mengeluarkan atau menggeser keluar): exactly same as pop, but it works for the first element in the array

Function Declaration vs Expression

identifier is a name from programming language to give names to variables, functions and objects

Function Declaration :

function identifier (parameterList(optional)) {
  functionBody
}

  • More Flexible, can written anywhere(before and after called function)
    • Because Hoisting concept

      • Hoisting is JavaScript mechanism where variables and function declarations are moved to the top of their scope before code execution

  • Tends to be easier to understand for beginners

Function Expression :

function identifier(optional) (parameterList(optional)) {
  functionBody
}

  • Must be defined first before being called
  • More Powerfull :
    • As closure
    • As an argument to another function
    • IIFE (Immediately Invoked Function Expression)

HOF (Higher Order Function

Higher Order Function is a function that operates an another function, whether it is used in arguments or as a return value

In javascript a function is called first class function

In javascript a function is treated as an object

Function as argument is called callback

An function that has a callback as its argument is called a Higher Order Function

Why use Higher Order Function :

  • Abstraction : make the code simple (hide the complexity in function)

Example Higher Order Function :

  • Array.prototype.map()
  • Array.prototype.filter()
  • Array.prototype.reduce()
